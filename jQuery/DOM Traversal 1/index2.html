The DOM

The DOM or The "Document Object Model" is a structural interface used in XML and HTML documents to display a webpage and its content in a tree like structure where every element, attribute and text node is considered an object, this modular approach allows deveopers to easily navigate and manipulate through content using a scripting language such as JavaScript.
